# FrontendTasks

A collection of frontend web development projects built using HTML, CSS, and JavaScript. This repository contains multiple mini-projects and practice applications designed to improve frontend development skills.

## Projects Included

### 1. Student Management System
A web-based application for managing student information.
- Add student details
- Update records
- Contact Us page
- Responsive design

### 2. Online Course Registration
An online course registration interface.
- Course listing
- Registration forms
- User-friendly UI
- Responsive layout

### 3. CLG Event Management System
A college event management website.
- Event information display
- Registration pages
- Contact Us section
- Modern frontend design

### 4. Personal Portfolio
A personal portfolio website showcasing:
- About Me section
- Skills
- Projects
- Contact Information
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/CNU2005/FrontendTasks.git
```

2. Open any project folder.

3. Run the `index.html` file in your browser.

## Purpose

This repository is created for practicing frontend development concepts, improving UI design skills, and building real-world web applications.
